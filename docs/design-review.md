# Design Review

- A first-time visitor can understand the offer: build a six-cookie box for Friday collection.
- Product quantity is clear: exactly six cookies per box.
- Pricing rule is shown before selection: a Chocolate Chip box is £15.99; a box with any Oreo or Biscoff is £18.49.
- The builder shows selected count, remaining count, breakdown, visual slots and live price.
- Purchase action is thumb-friendly on mobile with a sticky summary.
- Preorder timeline is explained on the homepage and How It Works page.
- Collection-only is visible before cart/checkout.
- Placeholder reviews are clearly labelled.
- Allergen copy is cautious and not definitive.
- Generated images are identified in docs as prototype-only.
- Every major feature is mapped in `shopify-transfer-guide.md`.
- Free apps are audited in `free-app-audit.md`.
- Remaining risk: fully tamper-proof exact-six validation cannot be guaranteed free with theme JavaScript alone.
