# Jake's Bakes Prototype

Static HTML/CSS/JS prototype for a premium home cookie preorder business in Barnsley.

## Open the prototype

Open `index.html` in a browser, or run a simple static server from this folder:

```powershell
python -m http.server 8080
```

Then visit `http://localhost:8080`.

## Key prototype paths

- `index.html` - homepage
- `build-a-box.html` - exact-six box builder
- `cart.html` - cart review and collection slot demo
- `faq.html` - launch FAQ with placeholders
- `policies/` - draft legal/customer information templates
- `docs/shopify-transfer-guide.md` - Shopify implementation map
- `docs/free-app-audit.md` - current free-app audit

## Prototype notes

The images are AI-generated visual direction only and must be replaced with real photographs of Jake's final cookies before launch. The policy and allergen copy is draft structure only, not legal advice.
