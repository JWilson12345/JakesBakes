# Free App Audit

Pricing checked on 16 July 2026 from Shopify App Store listings. Recheck before launch because app pricing changes.

## Summary Recommendation

Do not install apps unless they solve a confirmed launch problem. Start with native Shopify inventory, local pickup, Shopify notifications, theme pages, and manual approved testimonials.

Use a free pickup/time-slot app only if its free tier supports the final number of collection slots and reliable order capture. Avoid preorder apps for this model because native inventory and weekly manual opening are simpler.

## Native / Low-Code Alternative

For launch, use Shopify native inventory and a product-template selector:

- One product with Classic and Premium Mix variants.
- Theme UI enforces exact-six for normal customers.
- Selections saved as line-item properties.
- Weekly capacity controlled by inventory.
- Collection slot stored as cart attribute or order note.
- Jake reviews orders by collection slot, surname and order number before baking.

This is simpler than adding multiple apps and avoids relying on app free tiers that may not support the exact model.
